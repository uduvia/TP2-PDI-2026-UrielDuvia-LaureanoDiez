# TP2-PDI-2026-UrielDuvia-LaureanoDiez
Repositorio para el Segundo Trabajo Práctico de la materia "Procesamiento de Imágenes 1" de la carrera de TUIA en FCEIA, UNR, dictada durante el primer semestre de 2026.

El repositorio contiene un informe que explica la lógica de la resolución y funcionamiento de los ejercicios del trabajo práctico, acompañado de imágenes.

El código puede ser ejecutado a través de cualquier terminal python, menteniendo las rutas relativas entre la carpeta raíz y la carpeta que contiene las imágenes necesarias "imagenes". Es requerimiento tener Python 3.11.9 (idealmente Python 3.13, versión en la que se corrió correctamente el código) e instaladas las librerías OpenCV, Numpy y Matplot.


Link del repositorio con todos los trabajos prácticos resueltos en el cursado: https://github.com/laureanodiez/pdi1-2026
